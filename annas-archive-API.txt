
API Overview
An unOfficial simple Api for searching the popular Anna's archive database for books with options to download for free.

searchBook:

This endpoint returns a list of books based on some search parameters.

SearchTerm (q): eg title, author, ISBN.

Author (author): Optional Field for filtering results by author if title already known.

Source (source): Field for filtering results by Anna sources. Defaults to all if none specified. [libgenLi, libgenRs, zLibrary, internetArchive, uploads, nexusStc, duxiu, zLibraryChinese, magzDb, sciHub]

Page (page): Field for getting results by Anna pages. Defaults to 1 if not set.

Language (lang): Field for filtering results by language. Defaults to English if none specified. [en, fr, it, de, es etc].

Category (cat): Field for filtering results by category. Defaults to all if none specified. [fiction, nonfiction, magazine, comic, musicalscore, other, unknown].`

FileType (ext): Field for filtering results by file type. Defaults to all if none specified. [epub, pdf, mobi, azw3]

Sorting (sort): Field for sorting results. Defaults to most relevant if none selected [newest, largest, oldest, smallest, mostRelevant]

searchJournal:

This endpoint returns a list of books based on some search parameters.

SearchTerm (q): eg title, author, or ISBN.

Author (author): Optional Field for filtering results by author if title already known.

Source (source): Field for filtering results by Anna sources. Defaults to all if none specified. [libgenLi, zLibrary, uploads, nexusStc, magzDb, sciHub]

Page (page): Field for getting results by Anna pages. Defaults to 1 if not set.

Language (lang): Field for filtering results by language. Defaults to English if none specified. [en, fr, it, de, es etc].

Category (cat): Field for filtering results by category. Defaults to all if none specified. [fiction, nonfiction, magazine, comic, musicalscore, other, unknown].`

FileType (ext): Field for filtering results by file type. Defaults to all if none specified. [epub, pdf, mobi, azw3]

Sorting (sort): Field for sorting results. Defaults to most relevant if none selected [newest, largest, oldest, smallest, mostRelevant]

info:

This endpoint returns related information (such as descriptions, DOIs, ISBNs, etc.) for a book/journal.

Md5 (md5): from the selected book retrieved previously from the findBook requesti'
